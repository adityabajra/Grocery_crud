# Grocery_crud

Grocery Bud:
A simple shopping list application.

Description:
This project is a functional grocery list app to add, update and remove items. 

Technologies:
HTML & CSS: Custom layout using CSS Grid and background overlays.
jQuery: Handles the interactive list logic and icon toggles.

File Structure
index.html: Main page.
css/: Styles for the hero background, form, and items.
js/: Application logic and data handling.
